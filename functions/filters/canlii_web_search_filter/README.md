# ⚖️ CanLII Web Search Filter
A toggle filter that enables OpenAI Responses web search restricted to the CanLII legal information domain.

**Recommended manifold:** [`functions/pipes/openai_responses_manifold`](https://github.com/jrkropp/open-webui-developer-toolkit/tree/main/functions/pipes/openai_responses_manifold)

**Installation:**
Copy `canlii_web_search_filter.py` into **Admin ▸ Functions** in Open WebUI.
